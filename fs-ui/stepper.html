    属性

    参数 说明 类型 默认值
    v-model 绑定的值 String, Numer -
    step 步进值 String, Numer 1
    min 最小值 String, Numer 1
    max 最大值 String, Numer Infinity
    disabled 是否disable Boolen -
    disableInput 禁止手动输入 Boolen -
    readonly 是否readonly Boolen -
    background 带背景 Boolen -

    事件

    event 说明 返回值
    input 绑定值改变时触发 value
    on-change 绑定值改变时触发 value
