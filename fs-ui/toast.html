    参数

    参数 说明 类型 默认值
    type 提示类型，可选值为 success fail text String text
    overlay 遮罩层 Boolean false
    message 内容 String -
    duration 时间 Number 2000
    position 位置，可选值为 top middle bottom String middle
    value 是否显示toast Boolean true
