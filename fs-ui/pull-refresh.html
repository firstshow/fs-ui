    属性

    参数 说明 类型 默认值
    v-model 是否刷新中 Boolean -
    disabled 禁用下拉刷新 Boolean -
    loadingText 加载时的文字 String -
    duration 动画过渡时间 Number 500ms

    插槽

    插槽名 说明
    - 默认内容
    loading 加载时的提示

    事件

    event 说明 返回值
    refresh 处理刷新的事件 -
