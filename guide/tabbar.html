    Tabbar API

    参数 说明 类型 默认值
    v-model 当前选中项下标 Number -
    fixed 是否固定在底部 Boolean true
    z-index 定位元素层级 Number 1

    Tabbar Event

    事件名 说明 参数
    change 切换时触发 active: 当前选中项下

    TabbarItem API

    参数 说明 类型 默认值
    icon 图标名称 String -
    to 路由跳转名称 String -
