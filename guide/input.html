    属性

    参数 说明 类型 默认值
    v-model 绑定的值 String, Number -
    prefixIcon 前置图标 String -
    suffixIcon 后置图标 String -
    label 前置描述文字 String -
    clearable 显示清除按钮 Boolen false
    disabled 原生属性 Boolen false
    placeholder 原生属性 String -
    type 原生属性 String 'text', 'textarea', 'password','number','search' 'text'

    事件

    event 说明 参数
    on-focus focus 事件 返回值 event
    on-blur blur 事件 返回值 event
    input input 事件 返回输入值 value
    on-change change 事件 返回值 event
    on-input-change input-change 事件 返回值 event
    on-search type 为search时，点击后置搜索按钮 返回值 event
