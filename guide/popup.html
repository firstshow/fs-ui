    属性

    参数 说明 类型 默认值
    v-model 当前组件是否显示 Boolean false
    overlay 内容 String true
    position 位置，可选值为 top right bottom left String -
    transition 动画名称 String -
    close-on-click-overlay 点击遮罩层是后关闭 Boolean true
