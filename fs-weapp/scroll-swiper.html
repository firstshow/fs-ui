    Scroll-Swiper 滚动轮播

    使用指南

    • 在 app.json 或 index.json 中引入scroll-swiper组件
    {
  "usingComponents": {
    "fs-scroll-swiper": "path/components/scroll-swiper/index"
  }
}
    1
    2
    3
    4
    5
    • 本组件的使用需要结合template模板，可在项目根目录下创建template, 然后创建名为scroll-swiper的模板, 默认模板如下：
    <template name="scroll-swiper">
  <view class="scroll-view-item {{item}}"></view>
  <view class="scroll-view-item">{{index}}</view>
  <view class="scroll-view-item {{item}}"></view>
</template>
