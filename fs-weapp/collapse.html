    Collapse 折叠面板

    使用指南

    • 确保 components 下引入behavors/miniprogram-computed.js, icon组件, card、collapse、collapse-group组件
    • 在 app.json 或 index.json 中引入组件
    
{
  "usingComponents": {
    "fs-collapse": "../../components/collapse/index",
    "fs-collapse-group": "../../components/collapse-group/index"
  }
}
